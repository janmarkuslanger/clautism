## Quick Start (For Visionaries)

```bash
pip install -r requirements.txt
ollama pull qwen2.5 && ollama serve         # summon the feelings engine
cp data/people.example.json data/people.json # list everyone you'd otherwise forget
export TELEGRAM_BOT_TOKEN="123:ABC..."        # acquire a robot mouthpiece
python -m birthday_flow                        # become a wonderful person, instantly
```

Too scared to let it loose on real humans? Disrupt yourself first:

```bash
BIRTHDAY_DRY_RUN=1 python -m birthday_flow    # feelings, but only to your terminal
```
